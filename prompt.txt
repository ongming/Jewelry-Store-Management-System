Create a frontend UI for a Jewelry Store Management System using Java Spring Boot (Thymeleaf), HTML, and CSS.

Project Requirements:

* Use Spring Boot structure
* Place HTML files in: src/main/resources/templates/
* Place CSS files in: src/main/resources/static/css/
* Separate UI into two main parts:

  1. Guest (customer - no login)
  2. Admin/Staff (login required)

Folder Structure:

templates/
├── guest/
│   ├── home.html
│   ├── products.html
│   ├── product-detail.html
│   ├── cart.html
│   └── checkout.html
│
└── admin/
├── login.html
└── dashboard.html

static/css/
├── guest/
│   ├── home.css
│   ├── products.css
│   └── common.css
│
└── admin/
├── login.css
└── dashboard.css

========================================
GUEST UI REQUIREMENTS (Luxury Style)
====================================

Design style:

* Inspired by Dior, Chanel luxury brands
* Black, white, gold color palette
* Elegant serif fonts
* Minimalist layout
* Smooth hover animations

Pages:

1. home.html

* Hero banner with jewelry image
* Navbar: Home, Products, Contact
* Featured products section
* Category section
* Footer

2. products.html

* Grid layout products
* Filter sidebar:

  * Category
  * Price range
* Product card:

  * Image
  * Name
  * Price
  * Hover effect

3. product-detail.html

* Large image gallery
* Product name, price
* Description
* Material, weight
* Add to cart button

4. cart.html

* List products
* Quantity update
* Total price

5. checkout.html

* Guest form:

  * CustomerName
  * Phone
  * Email
  * ShippingAddress
* Payment method:

  * COD
  * Bank
  * Momo/VNPAY

========================================
ADMIN UI REQUIREMENTS (Login + Dashboard)
=========================================

1. login.html

* Centered login form
* Background: blurred jewelry image
* Fields:

  * Username
  * Password
* Button: Login
* Error message UI
* Luxury style (dark + gold)

2. dashboard.html

* Sidebar:

  * Dashboard
  * Products
  * Orders
  * Customers
  * Inventory
  * Supplier
  * Staff/Admin
* Topbar:

  * Profile
  * Logout
* Main content:

  * Stats cards (orders, revenue)
  * Recent orders table

========================================
TECHNICAL REQUIREMENTS
======================

* Use Thymeleaf syntax for dynamic data

* Use reusable components:

  * header
  * footer

* Link CSS properly:

  <link rel="stylesheet" th:href="@{/css/...}">

* Use clean, semantic HTML5

* Responsive design (mobile friendly)

========================================
GOAL
====

Generate clean, modern, luxury UI similar to high-end jewelry brands.
Code must be well-structured and separated into files as described.
